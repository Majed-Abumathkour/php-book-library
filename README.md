# PHP Book Library

A simple single-file PHP app for managing a personal book library.

## What the app does

- Displays a table of books
- Adds new books with validation
- Edits existing books
- Deletes books with confirmation
- Supports search and sorting
- Optionally shows cover thumbnails

## Tech

- PHP
- Bootstrap 5 (CDN)
- Session storage (no database)

## Run locally

1. Put the project folder inside `htdocs`.
2. Start Apache in XAMPP.
3. Open `http://localhost/<project-folder>/index.php`.
